# **App Name**: AaveBaseYield

## Core Features:

- Wallet Connection: Connect to user's wallet using Reown SDK and display connected address.
- Balance Display: Display user's ETH and USDC balances with real-time updates.
- Aave Deposit: Deposit ETH or USDC into Aave via a one-click interface, showing current APY and transaction status.
- Position Overview: Show deposited amount, current balance (principal + interest), and total earnings, with auto-refresh.
- Withdraw Funds: Withdraw ETH or USDC from Aave with a 'Max' button and transaction status updates.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to convey trust and innovation, reflecting the DeFi nature of the app.
- Background color: Light gray (#F5F7FA) for a clean, modern interface that ensures readability.
- Accent color: Soft teal (#64BDB4) for interactive elements and highlights, creating a subtle, engaging user experience.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and highly readable interface.
- Use minimalist, consistent icons for wallet connection, balance display, and transaction statuses.
- A clean, block-based layout with clear sections for wallet info, balances, deposit, and withdrawal actions.
- Subtle loading animations and transitions for transaction statuses to enhance user engagement.
